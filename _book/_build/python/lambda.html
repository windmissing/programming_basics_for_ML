lambda

将for\if\计算集中到一个lambda中

for ...:  
    if ...:
        do something

例如想要实现这样的效果：

A = [1,2,3,4,6]
B = []
for a in A:  # 遍历A
    if a %2 == 0: # 取其中的偶数
        B.append(a * 100)  # 把a*100放入B
B

[200, 400, 600]
# 用lambda一句话完成以上功能
B = [a*100 for a in A if a%2==0]
B

[200, 400, 600]
